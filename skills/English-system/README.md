# English System · 英语表达与思维系统

> **表达不分清使用场景 = 等于没学会。**

## 简介

这是一个专注于**场景化英语表达**的学习系统。从语料、口语对话与写作练习中，提炼明确使用场景的可复用表达，通过写作与精修训练英语思维。

## 核心特点

- **强制分轨**：每条表达必须标注 `Speaking` / `Writing` / `Both`
- **场景优先**：所有整理以「我什么时候能用」为第一优先级
- **双向训练**：写作（Writing Lab）+ 口语（Active Practice）
- **Obsidian 友好**：输出可直接保存为 Markdown 长期积累

## 使用方法

1. 输入 `GO` + 内容（语料/对话/写作/草稿）
2. 等待系统按标准结构输出
3. 保存到 Obsidian 仓库：`/English/System/English-System-YYYY-MM-DD.md`

## 输入标记

| 标记 | 含义 |
|------|------|
| `【】` | 重点内容 |
| `#topic` | 主题标签 |
| `??` | 不确定 |
| `✅` | 值得学习 |

## 输出结构

```
英语学习记录｜YYYY-MM-DD
├── 0) 今日核心能力
├── 1) 主题 × 使用场景
├── 2) 可复用表达库
│   ├── 2.1 口语专用 (Speaking)
│   ├── 2.2 写作专用 (Writing)
│   └── 2.3 通用表达 (Both)
├── 3) 回答/写作结构 (Frameworks)
├── 4) 主动练习
│   ├── Speaking Practice
│   └── Writing Practice
├── 5) 原文摘录
└── 6) Writing Lab
    ├── 写前思维引导
    ├── 作文修改（四层）
    └── 表达反向提炼
```

## 硬规则

1. **不标 Usage 的表达 = 不允许出现**
2. 口语和写作混用时，必须解释原因
3. 框架 > 单句 > 单词
4. 所有输出以「我下次能不能用」为判断标准
