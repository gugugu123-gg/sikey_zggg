# Idea Due Diligence · 个人想法尽调

> 这是一个把「我想不想做」转化为「值不值得我现在做」的决策引擎。

---

## 项目简介

Idea Due Diligence 是一个个人想法可行性分析系统，帮助你判断一个想法是否：

- 是否真实
- 是否可行
- 是否值得投入时间
- 是否适合当下的你

**核心逻辑**：七步尽调流程，强制收敛到「下一步最低成本行动」。

---

## 核心特性

### 七步尽调流程

```
┌─────────────────────────────────────────────────────┐
│               Idea Due Diligence 七步流程            │
├─────────────────────────────────────────────────────┤
│  Step 1  │  第一性原理    │  问题是否真实            │
│  Step 2  │  主要矛盾      │  卡点在哪里              │
│  Step 3  │  状态匹配      │  适不适合现在            │
│  Step 4  │  低成本启动    │  能不能先试              │
│  Step 5  │  长期价值      │  是否值得长期            │
│  Step 6  │  机会成本      │  是否最优选择            │
│  Step 7  │  可逆性        │  风险等级                │
└─────────────────────────────────────────────────────┘
```

### 核心原则

1. **先拆解问题，再谈行动** - 不被情绪、热情、案例带偏
2. **所有结论必须落到「下一步最低成本行动」**
3. **允许否定，允许放弃** - 理性止损不是失败
4. **对齐现有系统** - 与 Daily Review、TELOS 联动

## 安装

### 方法一：通过 Git 克隆

```bash
# 克隆到 Claude Code 的 skills 目录
git clone https://github.com/[你的用户名]/idea-due-diligence.git ~/.claude/skills/idea-due-diligence
```

### 方法二：手动安装

1. 下载本项目的 ZIP 文件或克隆到本地
2. 将 `idea-due-diligence` 文件夹复制到 Claude Code 的 skills 目录：
   - **macOS/Linux**: `~/.claude/skills/`
   - **Windows**: `%USERPROFILE%\.claude\skills\`

3. 确保文件夹结构如下：
   ```
   ~/.claude/skills/idea-due-diligence/
   ├── SKILL.md       # 技能定义文件
   └── README.md      # 说明文档（本文件）
   ```

### 验证安装

重启 Claude Code 或重新加载 skills 后，在对话中输入：

```
/idea-due-diligence
```

如果安装成功，该技能将被激活。

## 七步尽调详解

### Step 1：第一性原理

**问题**：这个想法最底层想解决什么问题？

**判断**：
- 问题真实
- 问题部分真实 / 被放大
- 问题不成立

### Step 2：主要矛盾

**问题**：当前阻碍这个想法成立的最大因素是什么？

**输出**：
> 主要矛盾 = xxx

### Step 3：状态匹配

**问题**：这个想法「吃状态」还是「抗状态」？

**判断**：
- 适合当下
- 可以尝试但需降级
- 明显不适合现在

### Step 4：低成本启动

**问题**：最低成本的验证方式是什么？

**要求**：
- ≤30 分钟
- 不花钱
- 不依赖他人
- 可随时停止

**输出**：
> MVP = xxx

### Step 5：长期价值

**问题**：这件事能沉淀什么长期资产？

**判断**：
- 有长期价值
- 仅短期尝试价值
- 深入意义不大

### Step 6：机会成本

**问题**：如果不做这个，最可能做什么？

**输出**：
> 是否值得占用当前资源：是 / 否

### Step 7：可逆性

**问题**：这是个可逆决策还是不可逆决策？

**分类**：
- 双向门（可快速试错）
- 单向门（需谨慎）

---

## 最终决策

| 决策 | 结果 | 对接规则 |
|------|------|----------|
| ✅ 允许进入 | 可进入 Daily Review · 明日调整<br>可注册为 TELOS · Project | 进入计划 |
| ⚠️ 暂缓 | 条件不成熟，等待时机 | 不进入，条件成熟后重新分析 |
| ❌ 否决 | 性价比过低 | 不进入，视为理性止损 |
